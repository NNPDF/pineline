# Introduction

A suite of tools to compute theory predictions for a PDF fit.

Included projects:

- [PineAPPL](https://github.com/N3PDF/pineappl)
- [EKO](https://github.com/N3PDF/eko)
- [yadism](https://github.com/N3PDF/yadism)
- [Hawaian Vrap](https://github.com/NNPDF/hawaiian_vrap)
- [Pineko](https://github.com/N3PDF/pineko)
- [Runcards](https://github.com/NNPDF/runcards)
- [Grids](https://github.com/NNPDF/pineapplgrids)
- [Old Grids](https://github.com/NNPDF/applgrids)
